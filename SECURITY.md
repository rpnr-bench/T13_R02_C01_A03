# Security Policy

## Supported use

Nmap Report Pipeline is intended for local development, documentation maintenance, and controlled review workflows.

## Scope

Only run scan helpers against local/demo targets explicitly documented in this repository. Do not point these utilities at public Internet hosts or third-party systems.

## Reporting

Open a security issue through the project maintainer channel with reproduction steps and the affected files. Avoid posting secrets or sensitive environment details in public comments.
